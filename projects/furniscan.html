Damage Estimation of Household Wooden Furniture Using Computer Vision and Deep Transfer Learning Approaches

  • Tech Stack: React, TypeScript, Tailwind CSS,Roboflow API, COHere API, Cohere, JSPDF, Vercel
  • Github URL: Project Link

The FurniScan AI web application is built on a modern, high-performance technology stack designed for real-time AI inference. The frontend utilizes React and TypeScript for a scalable, type-safe user interface, styled rapidly using Tailwind CSS. At its core, the application leverages the Roboflow API running the specialized rfdetr-nano object detection model to accurately detect and localize wooden furniture damages from user-uploaded images. For user support and guidance, a conversational assistant powered by Cohere is integrated, while jsPDF handles the generation of professional damage and cost analysis reports. The entire scalable application is deployed instantly and reliably using Vercel.

  • Damage Detection Model: Roboflow rfdetr-nano The rfdetr-nano model is the core engine for image analysis. It is part of the RF-DETR (Roboflow Detection Transformer) family, which is a state-of-the-art transformer-based object detection architecture.
  • Function: It takes an uploaded image of furniture and provides real-time inference by drawing bounding boxes around detected damages (cracks, scratches, dents) and assigning a class label to each.
  • Nano Variant Advantage: The "Nano" size is crucial because it offers an excellent balance between high speed (low latency) and high accuracy, making it perfect for a web application where users expect instant results without needing massive computational power.
  • Architecture: Unlike traditional models like YOLO that require a post-processing step called Non-Maximum Suppression (NMS), RF-DETR is an end-to-end model, which simplifies the pipeline and contributes to its speed and efficiency. It leverages deformable cross-attention mechanisms to focus selectively on relevant image features, improving detection quality even with cluttered backgrounds.
