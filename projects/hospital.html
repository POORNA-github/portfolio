Fully Functional Hospital Management System Web Application

  • Tech Stack: PHP, HTML, CSS, JavaScript, MySQL, CodeIgniter Framework
  • Github URL: Project Link

The Hospital Management System is a complete solution for managing operations, patient care, and staff across a hospital, developed using the reliable LAMP/XAMPP stack foundation. The core application logic is built using the PHP CodeIgniter Framework, which provides a structured Model-View-Controller (MVC) architecture, simplifying complex tasks like secure user authentication, role-based access control (for Administrators, Doctors, etc.), and database interactions.

The user interface is crafted using HTML for structure and CSS for styling, ensuring a clean and accessible dashboard that clearly presents information like earnings, appointments, and patient records. MySQL serves as the central database, storing all critical data, including patient demographics, billing records, and departmental information, while the XAMPP local server environment provides the necessary Apache server and PHP runtime to run the system during development. The system's expansive features cover all clinical and administrative needs, including Patient Management, Appointment Scheduling, Billing/Invoicing, and specialized modules for Laboratories and X-ray Report Management, ensuring all facets of hospital operations are digitized and integrated.
